p5js 繪圖套件應用

   簡介

  • p5js 是一個輔助進行「視覺化設計與創作」的 JavaScript 函示庫。
  • 官方網站：https://p5js.org/

   設計方式

  • 建立包含 p5js 的網頁 
<html>
	<head>
		<script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.8.0/p5.js"></script>
		<script>
			function setup(){
				createCanvas(600,400) ;
			}
			function draw(){
				background(0);
			}
		</script>
	</head>
	<body>
	</body>
</html>
  1. 引用 p5js 工具包 ➯
    <script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.8.0/p5.js"></script>

  2. 撰寫準備工作的地方 ➯
    function setup(){
         ...
    }

  3. 每一次更新畫面的地方，想像外面有一個『永不停止的重複執行』 ➯
    function draw(){
         ...
    }

  • for 和 『 變數 variable 』 的應用
    1. 設計一個變數「x」，來代表數字。就可以用程式去改變數字。
      2. var x = 20 ;
rect( x , 50 , 30 , 30 ) ;
    2. 用 for 重複，每畫一個，就把 x 往後移動 50
      3. var x = 20 ;
for(var i=1 ; i<=5 ; i++){   // 畫 5 個
	rect( x , 50 , 30 , 30 ) ;
	x = x + 50 ;  // 往後移動 50
}

應用演練（2） - 圖樣移動

  運作方式

  1. 設計一個變數「x」，來代表數字。就可以用程式去改變數字。
    特別注意，需要在 function 外，才能夠把每次 draw() 執行後的結果保留下來。
    2. var x = 0 ;
function setup(){
	// ... 其他程式
}
function draw(){
	// ... 其他程式
}
  2. 每次畫完，就把 x 往後移動 10
    3. var x = 0 ;
function setup(){
	// ... 其他程式
}
function draw(){
	// ... 其他程式

	rect( x , 50 , 30 , 30 ) ; // 畫圖形
	x = x + 10 ; // 往後移動 10

}
  3. 用 if( ❔ ) 判斷走到邊邊後，再把 x 變成左邊的座標。
