使用特性

  放資料的地方

忍者學院的學員寫程式紀錄每天練習忍術的次數。每一段時間，他們會把全部的次數加起來跟師父說。 如果次數很多，師父會很高興，小忍者就有機會得到師父的獎賞。

當小忍者要跟電腦要一個小空間來放數字時，這時候就要使用術語『var』來跟電腦索取。

  • 跟電腦拿倉庫放數字的小忍者

基礎語法

  要一個倉庫，把資料記下來

  • 用 var 開頭， 並且自己給一個 名稱（如：data、data01、abc）
    未來要拿資料才知道怎麼跟電腦說
    •     var data ;
    var data01;
    var ddd ;
  • 用   =   把右邊的東西放進左邊的倉庫
    • // 放數字
data = 17 ;
data01 = 1700 ;

// 放文字，文字要加兩撇包起來
abc = 'ABCㄅㄆㄇ';
  • 合在一起寫也可以
    • var price = 777 ;
var weather = '好天氣' ;
  • 重複放，會發生什麼事？ 之前的就被覆蓋了！！
    • var x ;
x=1;
x=2;
x=3;
// x 最後就是 3
  • 可以把一個倉庫放到另一個倉庫嗎？ 可以 
    • var a1 = 50 ;
var a2 ;
var a3 ;
a2 = a1 ;
a3 = a1 ;
// a1 , a2 , a3 都是 50


  取出資料

  • 可以使用 console.log( xxx ) 檢視倉庫資料內容
  • 用『名稱』拿出來用。如：相加起來再放到另一個倉庫。 
    • var a = 5 ;
var b = 10 ;
var c = a+b ;
var text = '計算結果' ;
console.log(a) ;
console.log(b) ;
cossole.log(text) ;
console.log(c) ;

應用方式

  進行倉庫資料的計算

  • 數字資料    加 +    減-    乘 *    除 / 
    • var a = 9 ;
var b = 3 ;
var x1 = a + b ;
var x2 = a - b ;
var x3 = a * b ;
var x4 = a / b ;
  • 文字資料    連起來 +    
    • var t1 = '今天' ;
var t2 = '昨天' ;
var s1 = '晴天' ;
var s2 = '陰天' ;
console.log(  t1 + '天氣是' + s1 ) ;
console.log(  t2 + '天氣是' + s1 ) ;
    // 和數字一起連結的例子
var a = 15 ;
var b = 7  ;
var m = a*b ;
console.log(  '每天零用錢' + a + '元，存了' + b + '天，總共存了' + m + '元' ) ;


  東西加進來

  • 數字資料：只用一個倉庫，把數字一直累積進倉庫。（加完再放回原來倉庫） 
    • var a = 0 ;
a = a + 1 ;
a = a + 2 ;
a = a + 3 ;
console.log('最後的 a 是 ' + a) ;
    無效的例子 
    var a = 0 ;
a + 1 ;
a + 2 ;
a + 3 ;
console.log('最後的 a 是 ' + a) ;
  • 文字資料：只用一個倉庫，把文字一直串起來 
    • var t1 = '今天' ;
t1 = t1 + '天氣' ;
t1 = t1 + '非常好' ;
console.log(  t1  ) ;
    無效的例子 
    var t1 = '今天' ;
t1 + '天氣' ;
t1 + '非常好' ;
console.log(  t1  ) ;
