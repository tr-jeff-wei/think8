情境說明 操作說明

雙子對抗賽

   競賽方法

  1. 競賽雙方選定主角： p1 p2
  2. 每回合進行一個指令動作（移動、攻擊、防禦）
  3. 優先擊敗對手，或是在比賽結束時，具備最高 hp 獲勝
  • 〔進階玩法〕每人可以自己設計 3 個函式工具，進行動作組合。
    每個函式工具內不可以超過 3 組動作。

操作說明

  步驟

  • 前往遊戲頁面： http://code-seed.click/lecture/js/Javascript_Playground2/FunctionGame.html
  • 開啟頁面 console ，兩位玩家輪流輸入如下語法進行比賽

  操作函式 

    // --- 設定由誰進行動作 --- 
setPlayer('p1')	;   // p1  or p2

// --- 移動 （不可以超過 5 步）--- 方位：U（上）D（下）L（左）R（右）
move( 'U' , 2 ) ;
move( 'D' , 5 ) ;
move( 'L' , 3 ) ;
move( 'R' , 5 ) ;

// --- 攻擊 （數字為攻擊距離，不可以超過 10 ）--- 方位：U（上）D（下）L（左）R（右）
// --- 攻擊越多，會耗損越多體力
fire( 'U' , 2 ) ;
fire( 'D' , 5 ) ;
fire( 'L' , 3 ) ;
fire( 'R' , 5 ) ;

// --- 防禦 （數字為可以抵擋子彈次數，最多 10）
defense( 5 ) ;

  程式範例

  • 玩家 1 要往右走 3 步
    • setPlayer('p1')	; 
move( 'R' , 3 ) ;
  • 玩家 2 向下方攻擊 7 格的距離
    • setPlayer('p2')	;
fire( 'D' , 7 ) ;
  • 利用函式，設計複雜的動作 ➩ 同時上下左右攻擊
    • // 設計函式工具
function fire_all(){
	fire( 'U' , 5 ) ;
	fire( 'D' , 5 ) ;
	fire( 'L' , 5 ) ;
	fire( 'R' , 5 ) ;
}

// 使用函式工具
fire_all() ;
  • 利用函式，設計複雜的動作 ➩ 往右下方，向走樓梯一樣移動
    • // 設計函式工具
function super_move(){
	move( 'R' , 3 ) ;
	move( 'D' , 3 ) ;
	move( 'R' , 3 ) ;
}

// 使用函式工具
super_move() ;
