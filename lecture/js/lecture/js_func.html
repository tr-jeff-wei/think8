語法結構

   簡介

  • 將多個任務放進一個「工具包」中，只要使用這個「工具包」就可以把多個任務一次執行。

   設計方式

  • 如何撰寫與應用「工具包」，主要分成兩個部分：
  1. 定義「工具包」內容。使用 function 、工具包名稱、（） 
    2. // tool_name ↣ 自己定義工具包的名稱
function tool_name(){
	 // 工具包要執行的程式 ..
	 // 工具包要執行的程式 ..
	 // 工具包要執行的程式 ..
}
  2. 使用「工具包」 
    3. tool_name() ;  // 使用第一次
tool_name() ;  // 使用第二次
tool_name() ;  // 使用第三次

   範例

  • 製作印出文字看板的「工具包」

基礎語法

  輸入訊息到小工具中

  • 輸入「1個」訊息，放在變數 t 中，丟進小工具 
    • // 要印出的文字，透過變數 t ，傳進小工具中
function say( t ){
	console.log( "老師說：" + t ) ;
}

say( "各位同學好～" ) ;
  • 輸入「2個」訊息，放在變數 p , t 中，丟進小工具
    • function say( p , t ){
	console.log( p + "說：" + t ) ;
}

say( "Jeff" , "上課囉～" ) ;
//  "Jeff"     ↣ 傳進 p
//  "上課囉～"  ↣ 傳進 t

  小工具丟出訊息

  • 運用 return 把要丟出的訊息送出 
    • // 輸入學號，取得學生姓名
function get_student_name ( n ){
	if( n==1 ){
		return "Jeff" ;
	}
	if( n==2 ){
		return "Mary" ;
	}
	if( n==3 ){
		return "Arial" ;
	}
}

// 取得姓名後，把姓名存在變數 name 中
var name = get_student_name(2) ;
console.log( "2 號是 "+name) ;

  會說話的函式

  1. 撰寫 say() 函式，把傳入的話，說指定的次數 
    2. // 第一個位置： text ➩ 放要說的話
// 第二個位置： n    ➩ 放重複的次數
function say( text , n ){
	for( var i=1 ; i<= n ; i++  ){
		console.log( text ) ;
	}
}

// === 下面是使用函式 ======
say( "good job !! "  ,  3 ) ;
  2. 撰寫 say_pretty() 函式，把傳入的話，前後加上裝飾的符號 
    3. // 第一個位置： text   ➩ 放要說的話
// 第二個位置： symbol ➩ 裝飾的符號
function say_pretty( text , symbol ){
	console.log( symbol +"  "+ text +"  "+ symbol) ;
}

// === 下面是使用函式 ======
say_pretty( "good job !! "  ,  "💎💎💎" ) ;
  3. 撰寫 say_box() 函式，把傳入的話，上下左右加上裝飾的符號 
    4. // 第一個位置： text   ➩ 放要說的話
// 第二個位置： symbol ➩ 裝飾的符號
function say_box( text , symbol ){
	console.log(symbol+symbol+symbol+symbol+symbol+symbol);
	console.log( symbol +"  "+ text +"  "+ symbol) ;
	console.log(symbol+symbol+symbol+symbol+symbol+symbol);
}

// === 下面是使用函式 ======
say_box( "good"  ,  "💎" ) ;

  猜數字函式

  1. 撰寫 guess() 函式，看看傳入的數字是否猜中 
    2. // 第一個位置： n  ➩ 傳入數字
function guess( n ){
	var x = 7 ;
	if( n == x ){
		console.log( "恭喜猜中了！" );
	}else if( n < x ){
		console.log( "再猜大一點～" ) ;
	}else{
		console.log( "再猜小一點～" ) ;
	}
}

// === 下面是使用函式 ======
